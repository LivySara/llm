# 变量命名

## 规则

1. **合法字符**：变量名由字母、数字和下划线构成，数字不能开头。

   > ⚠️ 这里说的"字母"指的是 Unicode 字符（万国码），理论上中文、日文、希腊字母等均可作为变量名字符，但 **强烈建议使用英文字母**。特殊字符（如 `!`、`@`、`#` 等）不能出现在变量名中。

2. **大小写敏感**：Python 是大小写敏感的编程语言，大写的 `A` 和小写的 `a` 是两个不同的变量。

3. **避免关键字和保留字**：变量名不要跟 Python 的**关键字**重名，尽可能避开 Python 的**保留字**。
   - **关键字**：在 Python 程序中有特殊含义的单词（如 `is`、`if`、`else`、`for`、`while`、`True`、`False` 等）
   - **保留字**：Python 语言内置函数、内置模块等的名字（如 `int`、`print`、`input`、`str`、`math`、`os` 等）

## 惯例

1. 变量名通常使用小写英文字母，多个单词用下划线连接（即 **snake_case** 风格）
2. 受保护的变量用单个下划线开头（如 `_protected_var`）
3. 私有的变量用两个下划线开头（如 `__private_var`）

# 变量的使用

## 类型检查与转换

在 Python 中可以使用 `type()` 函数对变量的类型进行检查。

可以通过 Python 内置的函数来改变变量的类型，下面是一些常用的和变量类型相关的函数：

| 函数 | 说明 |
|------|------|
| `int()` | 将一个数值或字符串转换成整数，可以指定进制 |
| `float()` | 将一个字符串（在可能的情况下）转换成浮点数 |
| `str()` | 将指定的对象转换成字符串形式，可以指定编码方式 |
| `chr()` | 将整数（字符编码）转换成对应的（一个字符的）字符串 |
| `ord()` | 将（一个字符的）字符串转换成对应的整数（字符编码） |

## 常用数据类型

Python 中最为常用的数据类型有以下几种：

### 1. 整型（int）

Python 中可以处理任意大小的整数，而且支持二进制（如 `0b100`，换算成十进制是 4）、八进制（如 `0o100`，换算成十进制是 64）、十进制和十六进制（`0x100`，换算成十进制是 256）的表示法。

```python
print(0b100)  # 二进制整数，输出 4
print(0o100)  # 八进制整数，输出 64
print(100)    # 十进制整数，输出 100
print(0x100)  # 十六进制整数，输出 256
```

### 2. 浮点型（float）

浮点数也就是小数，之所以称为浮点数，是因为按照科学记数法表示时，一个浮点数的小数点位置是可变的。浮点数除了数学写法（如 `123.456`）之外还支持科学计数法（如 `1.23456e2`，表示 123.456）。

```python
print(123.456)    # 数学写法，输出 123.456
print(1.23456e2)  # 科学计数法，输出 123.456
```

### 3. 字符串型（str）

字符串是以单引号或双引号包裹起来的任意文本，比如 `'hello'` 和 `"hello"`。

### 4. 布尔型（bool）

布尔型只有 `True`、`False` 两种值，要么是 `True`，要么是 `False`，可以用来表示现实世界中的"是"和"否"，命题的"真"和"假"，状况的"好"与"坏"，水平的"高"与"低"等等。如果一个变量的值只有两种状态，我们就可以使用布尔型。

## 运算符优先级

Python 中运算符的优先级如下表所示（从上到下优先级由高到低）：

| 运算符 | 描述 | 示例 |
|--------|------|------|
| `[]`、`[:]` | 索引、切片 | `lst[0]`、`lst[1:3]` |
| `**` | 幂 | `2 ** 3` → `8` |
| `~`、`+`、`-` | 按位取反、正号、负号 | `~0b1010` → `-11`、`+5`、`-5` |
| `*`、`/`、`%`、`//` | 乘、除、模、整除 | `3 * 4` → `12`、`10 / 3` → `3.333...`、`10 % 3` → `1`、`10 // 3` → `3` |
| `+`、`-` | 加、减 | `3 + 4` → `7`、`5 - 2` → `3` |
| `>>`、`<<` | 右移、左移 | `8 >> 2` → `2`、`8 << 2` → `32` |
| `&` | 按位与 | `0b1100 & 0b1010` → `0b1000` (8) |
| `^`、`|` | 按位异或、按位或 | `0b1100 ^ 0b1010` → `0b0110` (6)、`0b1100 | 0b1010` → `0b1110` (14) |
| `<=`、`>`、`>=` | 比较运算符 | `3 < 4` → `True`、`3 >= 3` → `True` |
| `==`、`!=` | 等于、不等于 | `3 == 3` → `True`、`3 != 4` → `True` |
| `is`、`is not` | 身份运算符 | `a is b`、`a is not b` |
| `in`、`not in` | 成员运算符 | `1 in [1, 2, 3]` → `True`、`1 not in [1, 2, 3]` → `False` |
| `not`、`and`、`or` | 逻辑运算符 | `not True` → `False`、`True and False` → `False`、`True or False` → `True` |
| `=`、`+=`、`-=`、`*=` 等 | 赋值运算符 | `a = 5`、`a += 3` → `a` 变为 `8` |

### 运算符示例

#### 1. 索引和切片

```python
lst = [0, 1, 2, 3, 4]
print(lst[0])      # 索引，输出 0
print(lst[1:3])   # 切片，输出 [1, 2]
print(lst[:3])     # 切片，输出 [0, 1, 2]
print(lst[::2])    # 步长为 2，输出 [0, 2, 4]
```

#### 2. 算术运算符

```python
# 幂运算
print(2 ** 3)    # 输出 8

# 乘除模整除
print(10 * 3)    # 输出 30
print(10 / 3)    # 输出 3.3333333333333335
print(10 % 3)    # 输出 1（余数）
print(10 // 3)   # 输出 3（整除）

# 加减
print(3 + 4)     # 输出 7
print(5 - 2)     # 输出 3
```

#### 3. 比较运算符

```python
print(3 < 4)     # 输出 True
print(3 > 4)     # 输出 False
print(3 <= 3)    # 输出 True
print(3 >= 4)    # 输出 False
print(3 == 3)    # 输出 True
print(3 != 4)    # 输出 True
```

#### 4. 逻辑运算符

```python
print(not True)            # 输出 False
print(True and False)      # 输出 False
print(True or False)       # 输出 True
print(3 < 4 and 5 > 2)   # 输出 True
```

#### 5. 赋值运算符

```python
a = 5
a += 3      # 等价于 a = a + 3
print(a)    # 输出 8

a *= 2      # 等价于 a = a * 2
print(a)    # 输出 16

a -= 5      # 等价于 a = a - 5
print(a)    # 输出 11
```

#### 6. 成员和身份运算符

```python
# 成员运算符
print(1 in [1, 2, 3])       # 输出 True
print(4 not in [1, 2, 3])   # 输出 True

# 身份运算符
a = [1, 2, 3]
b = [1, 2, 3]
c = a
print(a is c)           # 输出 True（同一对象）
print(a is b)           # 输出 False（不同对象，但值相同）
print(a == b)           # 输出 True（值相同）
```
